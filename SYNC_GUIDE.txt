IELTS TRAINER — SYNC GUIDE
==========================

PUBLISH (send your changes to GitHub)
--------------------------------------
1. Open terminal in the project folder
2. Run:
       npm run save

   That's it. It commits your latest questions and submitted answers,
   then pushes them to GitHub automatically.


RECEIVE (pull changes on another device)
-----------------------------------------
First time on a new device:
   1. Install Node.js (https://nodejs.org)
   2. Clone the repo:
          git clone https://github.com/samjahanirad/IELTS_Trainer.git
   3. Enter the folder:
          cd IELTS_Trainer
   4. Install dependencies:
          npm install
   5. Create a .env file with your Anthropic API key:
          ANTHROPIC_API_KEY=your_key_here
   6. Start the app:
          npm start
   7. Open browser at http://localhost:3000

Every time after that (to get the latest updates):
   1. Open terminal in the project folder
   2. Run:
          git pull
   3. Restart the app:
          npm start


WHAT GETS SYNCED
-----------------
  default/task1.json   — Task 1 questions and model answers
  default/task2.json   — Task 2 questions and model answers
  submissions.json     — Your submitted practice answers


WHAT STAYS LOCAL ONLY
----------------------
  node_modules/        — reinstalled via npm install on each device
  .env                 — your API key (never commit this)
